Reference Architecture

This repository collects opinionated, runnable reference stacks—useful as blueprints for layering web frontends, HTTP APIs, relational data, and deployment tooling in a way that mirrors production concerns.

Implementations

01 — Enterprise 3-tier

React, Go (Gin), PostgreSQL

A full-stack users CRUD demo with a Vite + React + TypeScript UI, a Go API using clean architecture (Gin, pgx, structured logging), and three PostgreSQL 16 instances with logical replication from a write primary to read-oriented replicas. Docker Compose and Kubernetes manifests are included.
